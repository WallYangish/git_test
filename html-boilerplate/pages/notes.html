Without further adieu, let's get started on learning about ordered and unordered lists.

Lists

Undordered Lists

Unordered lists have to be

  • Done within paragraph
  • Use UL for "unlisted"
  • Used for things that don't have to be listed in a particuluar order

With these parameters in place, a list with bullet points is created. Good for things like shopping lists, general to-do lists, and I'm sure some other stuff

Ordered Lists

Ordered lists are practically the same as unordered but you have to

  1. Use the OL tag instead of UL
  2. Type out the list remembering to list things with LI nested within the OL tags
  3. Think of more things to write to fill out the list

This will appear similarly to unordered lists, but now instead of bullet points we are presented with a numbered list.
